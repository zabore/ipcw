# Single-event example dataset in IPCW long format

The
[single_example_dat](https://zabore.github.io/ipcw/reference/single_example_dat.md)
dataset converted to counting-process (long) format with IPCW weights
appended, as produced by the data-preparation steps in the single-event
vignette (or equivalently by
[`get_ipcw_wgt()`](https://zabore.github.io/ipcw/reference/get_ipcw_wgt.md)).

## Usage

``` r
data(single_example_ipcw_dat)
```

## Format

A data frame in long format with one or more rows per subject and
columns including:

- id_nest:

  Nested simulation identifier.

- id:

  Subject identifier.

- true_time:

  True (latent) event time used for simulation diagnostics.

- tstart:

  Left endpoint of the time interval.

- tstop:

  Right endpoint of the time interval.

- delta:

  Event indicator for the current interval.

- censor:

  Censoring indicator for the current interval.

- x:

  Binary treatment indicator.

- W2:

  Ancillary biomarker covariate.

- inv_wgt:

  Estimated probability of remaining uncensored (inverse of the weight).

- wgt:

  Unstabilized IPCW weight (reciprocal of `inv_wgt`).
