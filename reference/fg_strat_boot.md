# Bootstrap Fine-Gray coefficients from stratified FG weights

A 500-by-3 matrix of bootstrap log sub-distribution hazard ratio
estimates for the effect of baseline PSA quartile (`z1`) on metastasis,
each row obtained by fitting a stratified-weight Fine-Gray model to one
bootstrap sample of the competing risks guided example data. Provided as
package data to avoid re-running the time-consuming bootstrap in
vignettes and examples. Use with
[`get_boot_pci_cr()`](https://www.emilyzabor.com/ipcw/reference/get_boot_pci_cr.md)
to obtain bootstrap-based confidence intervals.

## Usage

``` r
data(fg_strat_boot)
```

## Format

A numeric matrix with 500 rows (bootstrap replicates) and 3 columns
(`z1` levels 1, 2, and 3 vs. the reference level 0).

## Source

Generated by the authors by applying
[`get_ipcw_boot_cr()`](https://www.emilyzabor.com/ipcw/reference/get_ipcw_boot_cr.md)
with `B = 500` and `seed = 20240917` to a competing risks dataset
simulated via
`sim_data_cr(n = 500, censoring = "baseline", beta1 = log(1.5), beta2 = log(2.25), beta3 = log(3.4), p = 0.3)`
with `set.seed(9843)`, then fitting
[`fg_weighted_cr()`](https://www.emilyzabor.com/ipcw/reference/fg_weighted_cr.md)
with stratified weights
([`add_fg_weights_cr()`](https://www.emilyzabor.com/ipcw/reference/add_fg_weights_cr.md)
with `strat = "yes"`) to each bootstrap sample and extracting the
coefficient column.
