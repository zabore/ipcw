# Single-event example dataset

A simulated dataset used in the single-event IPCW guided example. It
represents a clinical scenario where the event of interest is
progression-free survival, the binary treatment covariate `x`
distinguishes two treatment arms, and `W2` is an informative censoring
covariate (analogous to an ancillary biomarker). Censoring depends on
`W2`, making standard Kaplan-Meier and Cox analyses potentially biased.

## Usage

``` r
data(single_example_dat)
```

## Format

A tibble with 500 rows and 5 columns:

- S:

  True (latent) event time.

- t:

  Observed time (minimum of `S` and the censoring time).

- delta:

  Event indicator: 1 = event occurred, 0 = censored.

- x:

  Binary treatment indicator: 0 = chemotherapy, 1 = TKI.

- W2:

  Continuous covariate that drives informative censoring (analogous to
  an ancillary biomarker).

## Source

Generated by the authors via
[`sim_data_SE()`](https://www.emilyzabor.com/ipcw/reference/sim_data_SE.md)
with `set.seed(20240429)` and default parameters.

## Details

The dataset was generated with the following parameters:

- \\n = 500\\, \\\alpha = 0.05\\, \\\sigma = 500\\, \\\beta =
  \log(0.25)\\, \\\lambda = 0.01\\, \\\phi = -5\\.

- `set.seed(20240429)`.

See the single-event vignette for the full data-generation code.
